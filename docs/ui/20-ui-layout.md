# UI Layout Model

## Default Workspace

- Header: identity + runtime pulse
- Left rail: session navigation
- Center: session pulse + protocol + turns
- Right rail: artifacts + memory + review
- Footer: interventions and follow-up

## Inspector / Dev Mode

Collapsed section containing:

- runtime endpoint
- per-role model routing
- low-level runtime toggles
- raw event list

Default workspace must stay collaboration-first.
