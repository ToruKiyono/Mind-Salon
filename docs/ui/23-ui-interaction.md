# UI Interaction Model

## Start and Continue

- Start creates a new session thread.
- Follow-up continues the same task/session context.

## Deliberation Controls

- Continue deliberation
- Request review
- Enter revise path
- Human intervention
- Replay round

## Playback Controls

- auto play on/off
- next stop
- show all
- reset

Playback controls are for visibility of collaborative reasoning, not execution semantics.

## Role Evolution Signals

Turn cards should surface:

- reflection
- correction
- strategy update

These signals must be traceable and bounded by role responsibility.
