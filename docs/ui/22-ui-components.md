# UI Components

## SessionPulse

Shows stage, round, active role, focus artifact.

## ProtocolRail

Displays canonical stage timeline and completion.

## FocusManuscript

Highlights current focus artifact.

## RoleTurnStream

Shows ordered role turns with:

- turn type (proposal/critique/challenge/revision/review)
- decision
- status (running/done)
- model/backend metadata
- reflection/correction/strategy-update signals

## ArtifactThread

Shows lineage-oriented artifact evolution.

## MemoryAside

Shows memory records relevant to current task.

## ReviewGate

Shows awaiting decision, decision role, revise capability, last decision.

## InterventionBar

Session actions and follow-up entry.
