# Mind Salon UI Principles

## Product Identity

Mind Salon UI is a collaborative reasoning workspace, not a chat shell or model control panel.

## Non-Negotiable UX Principles

1. Session is primary.
2. Protocol state is always visible.
3. Artifact evolution is more important than raw message logs.
4. Deliberation rounds and critique/challenge/revision are explicit.
5. Review gate is visible and actionable.
6. Memory signals carry across sessions.
7. Runtime and model controls belong to Inspector/Dev Mode.

## Workspace Outcome

The default page must answer:

- What stage/round is this session in?
- Who is currently driving?
- What artifact is evolving?
- What critique/challenge happened?
- Is review pending, approved, or revised?
